System Audio Setup

Visualize Spotify, Apple Music, or any app on macOS

1

Get a Virtual Audio Device

Use one of these tools to route audio:

Loopback Recommended SoundSource BlackHole Free
2

Set System Input

Open System Settings and configure the virtual device:

System Settings → Sound → Input

Select your Loopback or BlackHole device as the input source

3

Launch Animus

Start the visualizer and play some music. The audio will flow through!

Tip: Use the sensitivity slider to adjust visualization intensity
